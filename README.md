# PizzaParty

## Overview
PizzaParty is a Spring Boot application designed to manage pizza orders efficiently. It provides RESTful APIs for handling customers, orders, and menu items.


## Technologies Used
- **Java**
- **Spring Boot**
- **PostgreSQL**
- **Maven**
- **Docker**
